# Desktop Pet 桌面宠物

在桌面悬浮展示透明背景的宠物视频,自动轮播本地文件夹里的素材。

## 特性
- 透明、无边框、置顶、可拖动的桌面宠物窗口
- 右键菜单选本地文件夹,自动轮播里面的视频(一个播完切下一个,循环)
- 提供**绿幕视频**即可:程序首次导入时用内置 ffmpeg 自动抠绿幕、转成透明视频并缓存,之后播放零开销
- 右键菜单调尺寸(小/中/大),记住窗口位置与尺寸
- 系统托盘:显示/隐藏、开机自启开关、退出

## 运行
```bash
npm install
npm start
```

## 使用
1. 启动后右键桌面上的宠物(或右键系统托盘图标)
2. 「选择视频文件夹…」选中放有绿幕宠物视频的文件夹
3. 等待首次转换完成,宠物即出现在桌面
4. 用菜单调尺寸、显示/隐藏、设置开机启动

## 视频素材要求
- 纯绿幕背景(标准绿 `#00FF00` 效果最佳)
- 支持 .mp4 / .mov / .webm / .mkv / .avi
- 转换产物(透明 webm)缓存在用户数据目录,不修改原始文件

## 工作原理
- **主进程** (`src/main.js`):创建透明置顶窗口、系统托盘与右键菜单、读写配置、调用 ffmpeg 把绿幕转成带 alpha 的透明 webm 并缓存复用
- **渲染进程** (`src/renderer/`):只负责顺序循环播放透明 webm,完全不接触绿幕处理
- **绿幕 → 透明**:`chromakey` 滤镜抠绿 + `libvpx-vp9` 编码 `yuva420p`(`-auto-alt-ref 0` 保证 alpha 不丢)
- 源视频与缓存目录分离;按「文件名 + 修改时间」判定缓存,改动或新增才重新转换

## 开发
```bash
npm test   # 运行纯逻辑单测(config / video-scanner)
```

## 项目结构
```
src/
├── main.js              # 主进程:窗口/托盘/菜单/配置/转换编排
├── preload.js           # IPC 安全桥
├── config.js            # 配置读写
├── video-scanner.js     # 扫描 + 过滤 + 缓存判定
├── converter.js         # ffmpeg 绿幕 → 透明 webm
└── renderer/            # 透明播放器(index.html / style.css / renderer.js)
test/                    # 纯逻辑单测
assets/tray-icon.png     # 托盘图标
```
